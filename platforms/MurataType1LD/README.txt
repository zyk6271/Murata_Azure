--------------------------------------------
MurataType1LD - README
--------------------------------------------

Provider    : Murata
Description : Murata Type1LD evaluation board

Module
  Mfr     : Murata
  P/N     : MurataType1LD
  MCU     : STM32F412RGY6
  WLAN    : BCM43438 Wi-Fi
  Antenna : Single printed antenna (and in-line switched Murata MM8030 RF connectors)

EVB Features
  JLINK-JTAG debug interface
  USB-JTAG debug interface
  USB-serial UART interface
  Power supply : USB and/or external +5v
  Reset button
  Module current monitor
  Sensors/Peripherals
     - 2 x Buttons
     - 3 x LEDs
     - 1 x 16Mbit serial flash
  28-pin Expansion header
